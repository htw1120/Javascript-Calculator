Software Development Plan

Requirements Specification

  1. This assignment makes a dynamic calculator using python, HTML, CSS, Django, and Javascript
  2. There will be several HTML pages that will show various calculations.
  3. Only one page will give the option to do the calculations.
  4. There is a lot of data that comes from a lot of sources. All of the data will be housed in my assignment for the most part.
  5. The output will be a clean calculator complete with links to other pages
  6. I can't think of specific algorithms that I use. I don't think it will be too complicated.
